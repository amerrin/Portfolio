Pre-reqs: 
	1. Idle 2.7.3
	2. Pygame

There are two ways to run Manpak!:

1. Double click "manpak.py"
	This should prompt a console giving you the choice of (easy,normal,hard).
	These are 3 different levels. Choose any - play!

2. Open "manpak.py" in Idle(2.7) and run!
	Same as above
	This will also print whether you have "died" or "won"

*warning* - there is music, so mute your speakers if you don't wish to hear it
